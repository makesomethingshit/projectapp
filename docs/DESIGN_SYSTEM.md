# 디자인 시스템

이 파일은 기존 디자인 시스템 문서의 호환 진입점입니다. 현재 기준 디자인 문서는 [DESIGN.md](DESIGN.md)입니다.

## 핵심 방향

- 조용하고 우아하며 구조적인 프로젝트 작업실.
- 일반적인 관리자 대시보드 느낌을 피합니다.
- 편집 디자인처럼 여백과 타이포그래피 위계를 사용합니다.
- 임의의 그라디언트, 과한 그림자, 장식 효과를 피합니다.
- 기존 컴포넌트와 `variables.css`의 토큰을 우선 사용합니다.

## 스타일 파일 역할

- `variables.css`: 테마, 색, 폰트, 토큰.
- `layout.css`: 앱 셸, 상단 바, 패널 레이아웃.
- `components.css`: 프로젝트 목록, 상세 패널, 작업 카드 등 일반 컴포넌트.
- `modals.css`: 모달과 폼.
- `graph.css`: 그래프 기본 시각 요소.
- `graph-interactions.css`: 그래프 상호작용 상태와 전체 그래프 보정.
- `focus-widget.css`: 집중 위젯.

## 변경 전 확인

UI를 바꾸기 전 [DESIGN.md](DESIGN.md)와 [FRONTEND.md](FRONTEND.md)를 확인합니다. 기존 문서의 깨진 문자열은 현재 코드와 새 문서로 대체해 판단합니다.
